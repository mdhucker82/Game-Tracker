♜ Game Night Tracker

One-time setup: connect a free Supabase database so everyone shares live data. Takes about 5 minutes.

1

Create a free Supabase project

Go to supabase.com → Sign up free → New project. Pick any name & region. Wait ~1 min for it to provision.

2

Create the database tables

In your Supabase project, click SQL Editor in the left sidebar, paste the code below, and click Run:

create table games_list (
  id serial primary key,
  name text not null unique
);
create table players_list (
  id serial primary key,
  name text not null unique
);
create table game_log (
  id bigserial primary key,
  created_at timestamptz default now(),
  date text,
  game text,
  players text[],
  winners text[],
  win_type text,
  notes text
);
alter table games_list enable row level security;
alter table players_list enable row level security;
alter table game_log enable row level security;
create policy "read" on games_list for select using (true);
create policy "insert" on games_list for insert with check (true);
create policy "delete" on games_list for delete using (true);
create policy "read" on players_list for select using (true);
create policy "insert" on players_list for insert with check (true);
create policy "delete" on players_list for delete using (true);
create policy "read" on game_log for select using (true);
create policy "insert" on game_log for insert with check (true);
create policy "delete" on game_log for delete using (true);
3

Enter your Supabase credentials

In Supabase go to Project Settings → API. Copy the Project URL and anon public key and paste them below.

4

Set your editor PIN

Only people with this PIN can log games and edit lists. Everyone else sees data read-only.
